Mojuri – Jewelry Store HTML Template



Overview

  • Created: 30/04/2023
  • Updated: 30/04/2023
  • By: CakeTheme
  • Email: caketheme.com@gmail.com

Thank you for purchasing my product. If you have any questions that are beyond the scope of this help file, please feel free to email via my user page contact form here. Thanks so much!

Mojuri is a clean, modern and impressive HTML template which fits for your jewelry store. It is an outstanding HTML based web template with a sleek and minimal design.

Mojuri has a fully responsive layout. It fits perfectly on various displays and resolutions from regular desktop screens to tablets, iPads, iPhones and small mobile devices. The template contains 32 HTML pages with 8 Homepages.

You can use this beautiful template. There is no doubt that Mojuri will make your work look more impressive and attractive to viewers.

Files Structure

  • /HTML
    • /assets (CSS and JS files)
    • /media (All Images)
    • /libs (All external libs. We keep all of them in this folder to make updates easily.)

Javascript

It's possible to change the JS settings by extending the defaults from the template, that way is so much easier to update the template changing only one JS file.
Here are a few samples on how to do that:

Changing Settings


1. Scroll to top (assets/js/app.js)
$('.back-top').on('click', function() {
    $('html, body').animate({
        scrollTop: 0
    }, 800);
})

2. Price filter (assets/js/app.js)
if ($('#price-filter').length) {
    $('#price-filter').slider({
        from: 0,
        to: 100,
        step: 1,
        smooth: true,
        round: 0,
        dimension: ' $',
        skin: 'plastic'
    });
}
